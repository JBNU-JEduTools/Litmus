# Litmus  
### 프로그래밍 과제 자동 채점 시스템

Litmus는 SW 교육을 위한 JEduTools 의 일환으로,
프로그래밍 과제의 자동 채점을 수행하는 시스템입니다.


전북대 컴퓨터공학부에서 개발하여 20여년간 활용해왔으며, 
2024년 오픈소스(DMOJ) 를 기반으로 새롭게 리뉴얼하였습니다.
